###USED TECHNOLOGIES
**React.Js
**Node.Js
**PostgreSQL DB
**Firebase storage for images

##Features
**Create user and update profile pic,username and bio
**User authentication
**Like Post,Comment on Post,Like Comment,Follow,DELETE post ,ADD post.
**Search for users
**Get notifications for follow,like and comment.
